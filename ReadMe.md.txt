# Qwen 2.5 Local AI Chatbot

This project is a local AI chatbot built using **Qwen 2.5**, **Ollama**, **FastAPI**, and **React**.

## Features
- Runs Qwen 2.5 locally using Ollama
- FastAPI backend for AI inference
- React-based chat UI
- No cloud APIs (100% local & private)

## Tech Stack
- Python
- FastAPI
- React (Vite)
- Ollama
- Qwen 2.5
